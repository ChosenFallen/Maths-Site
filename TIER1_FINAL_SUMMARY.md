## Distractor Quality

The `generateNumericDistracters()` function provides high-quality wrong answers:

### Patterns Used
1. **Off-by-1 errors**: ±1 from correct answer
2. **Scaling errors**: Doubled, halved, or 50% more
3. **Sign errors**: Negative of correct answer
4. **Magnitude errors**: ±10 variations

### Example
- Correct: **24**
- Distractors: 23, 25, 48, 12, -24, 36, 34, 14...

Each question gets 3 randomly selected distractors from these patterns.
