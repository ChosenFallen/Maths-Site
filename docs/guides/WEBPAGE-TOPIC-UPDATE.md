## Recommended Approach for Webpage

**Option A: Keep Current Names but Reorganize**
- Keep the 23 topic names (they're actually quite good)
- Group them visually under the 6 GCSE categories
- Add GCSE code references in metadata

**Option B: Rename to Official Subtopic Names**
- Replace "Algebra & Equations" with "Solving Equations"
- Replace "Functions & Graphs" with "Graphs and Functions"
- Etc. - More formally aligned with curriculum

**Option C: Hybrid Approach (RECOMMENDED)**
- Use main GCSE category as primary
- Keep descriptive subtopic name as secondary
- Example: "Algebra: Equations and Inequalities"
