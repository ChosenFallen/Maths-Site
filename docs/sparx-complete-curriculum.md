# Sparx Mathematics Curriculum Mapping

Purpose: Complete mapping of Sparx curriculum topics with prerequisites and progression paths for Do Now question selection.

Structure: 
- Each topic lists its building blocks (prerequisites from earlier in curriculum)
- Each topic shows further topics (where this content leads)
- Teachers select a topic and Do Now automatically pulls questions from:
  1. Building blocks for the selected topic
  2. Previous topic (revision)
  3. Topic before that (revision)
  4. A random earlier topic

## Implementation Notes

### Do Now Question Selection Algorithm

When a teacher selects a topic:

1. **Question 4 (Building Blocks)**: 
   - Fetch random question from worksheets tagged with this topic's building blocks
   
2. **Question 3 (Last Topic - Revision)**:
   - Fetch random question from worksheets tagged with the immediately preceding topic
   
3. **Question 2 (Two Topics Back - Revision)**:
   - Fetch random question from worksheets tagged with the second-previous topic
   
4. **Question 1 (Earlier Topic)**:
   - Fetch random question from any earlier topic (can span year groups)

### Linking Worksheets to Topics

Once you provide the Sparx resource mappings, we will:
1. Tag each worksheet with Sparx topic codes
2. Create question banks per topic/difficulty level
3. Store building block relationships
4. Enable teachers to select topics from this curriculum
