## How to Use This Reference

1. **Find the topic** you're teaching in the sections above
2. **Note the Sparx Topic Codes** (M###) - these match our curriculum system
3. **Refer to the PDFs** in the `Sparx-files/` folder for complete question banks
4. **Use with Do Now** - Topic codes link to our topic selector

## File Types Explained

- **BuildingBlocks**: Foundation-level questions targeting prerequisite skills
- **ExamplePairs**: Worked examples with teacher and student versions
- **Deepen**: Extension and enrichment questions for deeper understanding
- **Assessment**: Formal assessment items with mark schemes
- **Revision**: Mixed topic revision materials

## Integration with Our System

Each Sparx unit maps to:
- Specific topic codes (M###) in our curriculum
- Corresponding worksheets from our 67-worksheet library
- Building blocks that inform the Do Now question selection
