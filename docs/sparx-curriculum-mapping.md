# Sparx Curriculum Mapping for Do Now Questions

**Purpose**: Map each Sparx topic to:
- Building blocks (prerequisite skills needed)
- Further topics (what this topic leads to)
- Recommended worksheet questions for each skill area

**Do Now Selection Logic**:
When a teacher selects a topic:
1. **Question 4 (Building Blocks)**: Random question from building blocks for this topic
2. **Question 3 (Last 2 Topics - Revision)**: Random question from the immediate previous topic
3. **Question 2 (Last 2 Topics - Revision)**: Random question from the second previous topic
4. **Question 1 (Earlier Revision)**: Random question from any earlier topic (can span to previous year groups)
