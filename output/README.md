# Generated Output Files

This folder contains generated content and exported materials.

## 📄 Contents

### [pdfs/](pdfs/)
Generated PDF files for worksheets (60+ files)
- One PDF per worksheet
- Generated via `npm run generate-pdfs`
- Can be printed or distributed to students

**To generate/update PDFs:**
```bash
npm run generate-pdfs           # All worksheets (~20 min)
npm run generate-pdfs [id]      # Single worksheet (~30 sec)
```

### [exam-papers/](exam-papers/)
GCSE exam paper series and documentation
- Series 1: 6 complete papers (3 Foundation, 3 Higher)
- Answer schemes and marking guides
- **Start here:** [exam-papers/QUICK-START-PDF.md](exam-papers/QUICK-START-PDF.md)

### papers-analysis.html
Interactive visualization tool for analyzing past papers:
- Browse all 54 official GCSE papers (2017-2024)
- Filter by year, paper type, and topics
- View topic distribution charts
- Open in a web browser

## 🔄 How These Files Are Generated

| File/Folder | Generated By | Purpose |
|---|---|---|
| `pdfs/` | `npm run generate-pdfs` | Distributable worksheet PDFs |
| `exam-papers/*.pdf` | [convert-to-pdf.sh](exam-papers/convert-to-pdf.sh) | Exam paper PDFs (needs Pandoc) |
| `papers-analysis.html` | Past paper analysis tool | Interactive past paper browser |

## 📝 Notes

- **pdfs/** is in `.gitignore` (generated, not version controlled)
- **exam-papers/** contains both source (markdown) and output (PDFs)
- All PDFs maintain professional formatting suitable for classroom use
- Files are automatically organized by type
